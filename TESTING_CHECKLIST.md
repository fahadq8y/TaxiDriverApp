## ✅ السيناريوهات المطلوب اختبارها

### 1. التتبع الأساسي
- [ ] التطبيق يبدأ التتبع تلقائياً بعد تسجيل الدخول
- [ ] المواقع تُحفظ في Firestore بشكل صحيح
- [ ] الدقة جيدة (accuracy < 20 متر)
- [ ] التحديثات منتظمة (كل 10-30 ثانية)

### 2. التتبع في الخلفية
- [ ] التتبع يستمر عند قفل الشاشة
- [ ] التتبع يستمر عند فتح تطبيقات أخرى
- [ ] التتبع يستمر عند استخدام الهاتف لساعات
- [ ] الإشعار خفي (غير مزعج)

### 3. إعادة التشغيل
- [ ] التتبع يبدأ تلقائياً بعد إعادة تشغيل الجهاز
- [ ] البيانات محفوظة (لا يحتاج تسجيل دخول)
- [ ] التتبع يعمل فوراً بعد التشغيل

### 4. Force Stop (الاختبار الأهم)
- [ ] افتح إعدادات الجهاز → التطبيقات → TaxiDriverApp
- [ ] اضغط "Force Stop" أو "إيقاف إجباري"
- [ ] انتظر 5 دقائق
- [ ] هل وصلت إشعارات FCM؟
- [ ] هل بدأ التتبع تلقائياً؟
- [ ] أو يحتاج فتح التطبيق يدوياً؟

### 5. البطارية
- [ ] استهلاك البطارية معقول (< 10% في 8 ساعات)
- [ ] التطبيق لا يسخن الجهاز
- [ ] Battery Optimization معطل للتطبيق

### 6. الإنترنت
- [ ] التتبع يعمل مع 4G/5G
- [ ] التتبع يعمل مع WiFi
- [ ] ماذا يحدث عند فقدان الإنترنت؟
- [ ] هل تُحفظ المواقع محلياً؟
- [ ] هل تُرسل بعد عودة الإنترنت؟

### 7. تسجيل الخروج
- [ ] التتبع يستمر بعد Logout
- [ ] يمكن تسجيل الدخول مرة أخرى
- [ ] البيانات صحيحة بعد إعادة الدخول

### 8. الاستقرار
- [ ] التطبيق لا يتوقف (No Crashes)
- [ ] التطبيق لا يتجمد (No Freezes)
- [ ] WebView يعمل بشكل صحيح
- [ ] التطبيق يعمل لأيام بدون مشاكل
